1. Project Overview

Tripzverse.in is a modern, feature-rich travel booking platform that combines stunning visual design with powerful functionality. Built with HTML, CSS, and JavaScript, this platform offers a seamless user experience for booking hotels, buses, trains, and local transportation.

Key Highlights

  • AI Travel Guide: Integrated assistant for destination information
  • Modern Design: Purple-violet glassmorphism aesthetic
  • Responsive: Works flawlessly on all devices
  • Theme Support: Light and dark mode with persistence
  • Lightweight: Pure vanilla JavaScript, no heavy frameworks

4. Setup & Installation

Prerequisites

  • A modern web browser (Chrome, Firefox, Safari, Edge)
  • A Gemini API key (for AI features)

Installation Steps

  1. Download/Clone the Repository 
    git clone https://github.com/yourusername/tripzverse-travel.git
  2. Open the Project

    Simply open index.html in your web browser, or use a local server:

    # Using Python
python -m http.server 8000

# Using Node.js
npx http-server
  3. Configure API Key (Optional)

    Open gemini.js and replace the API key with your own.

üí° Tip: For the best experience, use a local server rather than opening the HTML file directly.

5. File Descriptions

index.html

The main HTML structure containing all page sections:

  • Navigation bar with responsive hamburger menu
  • Hero section with animated search tabs
  • Hotels section with filterable cards
  • Buses section with route listings
  • Trains section with scenic routes
  • Local transport with fare calculator
  • World destinations gallery
  • AI Assistant chat interface
  • Footer with newsletter signup

styles.css

Complete design system including:

  • CSS custom properties (variables) for theming
  • Light and dark mode definitions
  • Glassmorphism effects with backdrop blur
  • Responsive layouts and media queries
  • Animation keyframes

script.js

Core functionality modules:

  • initTheme() - Theme toggle with localStorage
  • initNavigation() - Scroll effects & mobile menu
  • initSearchTabs() - Tab switching logic
  • initCounterAnimation() - Stats counter animation
  • initScrollAnimations() - Scroll-triggered effects
  • initHotelFilters() - Hotel category filtering
  • initFareCalculator() - Transport fare estimation

gemini.js

AI assistant integration:

  • API connection & configuration
  • Chat UI management & message rendering
  • Markdown formatting support
  • Typing indicators & animations

6. Customization Guide

Changing Theme Colors

Edit the CSS variables in styles.css:

:root {
    --primary: #8b5cf6;        /* Main purple */
    --primary-light: #a78bfa;  /* Light purple */
    --primary-dark: #7c3aed;   /* Dark purple */
    --secondary: #c084fc;      /* Secondary violet */
    --accent: #a855f7;         /* Accent color */
}

Adding New Destinations

Add new destination cards in index.html:

<div class="destination-card" data-continent="asia">
    <div class="dest-image">
        <img src="image-url" alt="City Name">
        <div class="dest-overlay">
            <span class="dest-country">Country</span>
            <h3>City Name</h3>
        </div>
    </div>
</div>

Adding New Hotels

Copy an existing hotel card and modify the content. Available categories:

  • luxury - Luxury hotels
  • boutique - Boutique hotels
  • resort - Resort properties
  • budget - Budget-friendly options

7. API Configuration

Getting an API Key

  1. Visit Google AI Studio
  2. Sign in with your Google account
  3. Navigate to "Get API Key"
  4. Create a new API key

Configuring the API

Open gemini.js and update line 6:

const GEMINI_API_KEY = 'your-api-key-here';
‚ö†Ô∏è Security Note: Never commit your API key to public repositories. Consider using environment variables or a backend proxy for production deployments.

Customizing Responses

The system prompt in gemini.js defines how the assistant responds. You can modify it to:

  • Change the response format
  • Add or remove information categories
  • Adjust the response length
  • Modify the assistant's tone
